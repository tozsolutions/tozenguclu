                    OPENAI API KEY KURULUM REHBERİ
              GitHub + Netlify + Render Entegrasyonu

🔑 ADIM 1: OPENAI API KEY ALMA
═══════════════════════════════════════════════════════════════════

1. 🌐 https://platform.openai.com adresine git
2. 🔐 "Sign up" ile hesap oluştur (Gmail ile hızlı)
3. 📧 Email doğrulama yap
4. 💳 Kredi kartı bilgilerini ekle (ilk $5 ücretsiz)
5. 🔑 Sol menüden "API keys" sekmesine tıkla
6. ➕ "Create new secret key" butonuna tıkla
7. 📝 İsim ver: "ModernSite-Production"
8. 📋 Key'i kopyala (sk-proj-... ile başlar)
9. ⚠️  ÖNEMLİ: Bu key'i güvenli bir yere kaydet!

═══════════════════════════════════════════════════════════════════
🐙 ADIM 2: GITHUB REPOSITORY OLUŞTURMA
═══════════════════════════════════════════════════════════════════

1. 🌐 https://github.com adresine git
2. ➕ "New repository" tıkla
3. 📝 Repository adı: "modernsite-ai-backend"
4. ✅ "Public" seç (ücretsiz plan için)
5. ✅ "Add a README file" işaretle
6. 🎯 "Create repository" tıkla

📁 BACKEND DOSYALARINI YÜKLEME:
1. 💻 Bilgisayarınızda "modernsite-ai-backend" klasörü oluştur
2. 📂 Aşağıdaki dosyaları kopyala:
   - backend/app.py
   - backend/requirements.txt
   - backend/render.yaml
3. 🔄 GitHub'a yükle:
   - "uploading an existing file" tıkla
   - Dosyaları sürükle-bırak
   - "Commit changes" tıkla

═══════════════════════════════════════════════════════════════════
🚀 ADIM 3: RENDER'DA BACKEND DEPLOY
═══════════════════════════════════════════════════════════════════

1. 🌐 https://render.com adresine git
2. 🔐 "Get Started for Free" tıkla
3. 🐙 "GitHub" ile giriş yap
4. ➕ "New +" > "Web Service" seç
5. 🔗 "Connect GitHub" tıkla
6. 📂 "modernsite-ai-backend" repository'yi seç
7. ⚙️ AYARLAR:
   - Name: modernsite-ai-backend
   - Environment: Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: gunicorn app:app
   - Instance Type: Free ($0/month)

8. 🔐 ENVIRONMENT VARIABLES:
   - "Advanced" sekmesine tıkla
   - "Add Environment Variable" tıkla
   - Key: OPENAI_API_KEY
   - Value: sk-proj-... (OpenAI'dan aldığınız key)
   - "Add Environment Variable" tekrar tıkla
   - Key: FLASK_ENV
   - Value: production

9. 🚀 "Create Web Service" tıkla
10. ⏳ 5-10 dakika bekle (deployment)
11. ✅ URL'i kopyala: https://modernsite-ai-backend.onrender.com

═══════════════════════════════════════════════════════════════════
🌐 ADIM 4: NETLIFY'DA FRONTEND DEPLOY
═══════════════════════════════════════════════════════════════════

🎯 VERSİYON 1 (BASIC - AI'SIZ):
1. 🌐 https://netlify.com adresine git
2. 🔐 "Sign up" tıkla (GitHub ile giriş)
3. 📁 "Sites" > "Add new site" > "Deploy manually"
4. 🗂️ basic-version/ klasörünü sürükle-bırak
5. ⏳ 2-3 dakika bekle
6. ✅ Site hazır: https://random-name.netlify.app

🤖 VERSİYON 2 (AI'LI):
1. 📝 ÖNCE API URL'İNİ GÜNCELLE:
   - ai-version/chatbot.js dosyasını aç
   - Line 8: apiEndpoint'i değiştir
   - ESKI: 'https://your-render-app.onrender.com/api/chat'
   - YENİ: 'https://modernsite-ai-backend.onrender.com/api/chat'

2. 🌐 Netlify'da "Add new site" > "Deploy manually"
3. 🗂️ ai-version/ klasörünü sürükle-bırak
4. ⏳ 2-3 dakika bekle
5. ✅ Site hazır: https://random-name-2.netlify.app
