                    2. BACKEND DEPLOYMENT (RENDER)
═══════════════════════════════════════════════════════════════════

🐙 GITHUB KURULUMU:
1. https://github.com git
2. "New repository" tıkla
3. Repository adı: "modernsite-ai-backend"
4. "Public" seç
5. backend/ klasöründeki dosyaları yükle:
   - app.py
   - requirements.txt
   - render.yaml

🚀 RENDER DEPLOYMENT:
1. https://render.com git
2. "Get Started for Free" tıkla
3. GitHub ile giriş yap
4. "New +" → "Web Service"
5. "Connect GitHub" tıkla
6. "modernsite-ai-backend" repository seç

⚙️ RENDER AYARLARI:
- Name: modernsite-ai-backend
- Environment: Python 3
- Build Command: pip install -r requirements.txt
- Start Command: gunicorn app:app
- Instance Type: Free ($0/month)

🔐 ENVIRONMENT VARIABLES:
1. "Advanced" sekmesine tıkla
2. "Add Environment Variable":
   - Key: OPENAI_API_KEY
   - Value: sk-proj-your-actual-key
3. "Add Environment Variable":
   - Key: FLASK_ENV
   - Value: production

🚀 DEPLOY:
1. "Create Web Service" tıkla
2. 5-10 dakika bekle
3. URL'i kopyala: https://modernsite-ai-backend.onrender.com
4. ✅ Backend hazır!
