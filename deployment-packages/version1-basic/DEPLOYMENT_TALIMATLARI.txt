═══════════════════════════════════════════════════════════════════
                    1. NETLIFY DEPLOYMENT
═══════════════════════════════════════════════════════════════════

🌐 ADIMLAR:
1. https://netlify.com adresine git
2. "Sign up" tıkla (Gmail ile hızlı kayıt)
3. Ana sayfada "Add new site" tıkla
4. "Deploy manually" seç
5. Bu klasördeki TÜM dosyaları seç ve sürükle-bırak
6. 2-3 dakika bekle
7. ✅ Site hazır!

🔧 AYARLAR:
- Site adı değiştir: "yourcompany-basic"
- Custom domain ekleyebilirsin
- HTTPS otomatik aktif
- Form submissions otomatik çalışır

📊 SONUÇ URL: https://yourcompany-basic.netlify.app

═══════════════════════════════════════════════════════════════════
                    2. GITHUB + NETLIFY AUTO-DEPLOY
═══════════════════════════════════════════════════════════════════

🐙 GITHUB KURULUMU:
1. https://github.com git
2. "New repository" tıkla
3. Repository adı: "basic-website"
4. "Public" seç
5. Bu klasördeki dosyaları yükle

🔗 NETLIFY BAĞLANTISI:
1. Netlify'da "Add new site" → "Import an existing project"
2. "GitHub" seç
3. Repository'yi bağla: "basic-website"
4. Build settings:
   - Build command: (boş bırak)
   - Publish directory: . (nokta)
5. "Deploy site" tıkla

✅ AVANTAJ: Her GitHub push'ta otomatik güncellenir!

═══════════════════════════════════════════════════════════════════
                    3. GÜZEL HOSTING DEPLOYMENT
═══════════════════════════════════════════════════════════════════

🌐 GÜZEL HOSTING KURULUMU:
1. https://guzelhosting.com.tr git
2. cPanel'e giriş yap
3. "File Manager" aç
4. "public_html" klasörüne git
5. Bu klasördeki dosyaları yükle
6. .htaccess dosyasını aktif et

⚙️ AYARLAR:
- SSL sertifikası aktif et
- Domain ayarları yap
- Email hesapları kur

📧 FORM AYARLARI:
- PHP mail() fonksiyonu kullan
- SMTP ayarları yap (isteğe bağlı)
