                       YÜKLEME KLAVUZU
═══════════════════════════════════════════════════════════════════

📋 GEREKSİNİMLER:
- Web hosting hesabı (Apache/Nginx)
- SSL sertifikası (önerilen)

- FTP/SFTP erişimi

🔧 KURULUM ADIMLARI:

1. 📁 DOSYA YÜKLEME:
   - Tüm dosyaları web sunucunuzun root dizinine yükleyin
   - Dosya izinlerini kontrol edin (644 dosyalar için, 755 klasörler için)
   
   Yüklenecek dosyalar:
   ├── index.html          (Ana sayfa - 19KB)
   ├── styles.css          (Stil dosyası - 24KB)
   ├── script.js           (JavaScript - 16KB)
   ├── sw.js              (Service Worker - 1.4KB)
   ├── manifest.json      (PWA manifest - 743B)
   ├── sitemap.xml        (SEO sitemap - 1KB)
   ├── robots.txt         (Arama motoru - 228B)
   ├── .htaccess          (Apache config - 1.8KB)
   ├── 404.html           (Hata sayfası - 2.5KB)
   ├── favicon.svg        (Site ikonu - 251B)
   └── README.md          (Dokümantasyon - 2.1KB)

2. 🌐 DOMAIN YAPILANDIRMASI:
   - index.html, sitemap.xml ve manifest.json dosyalarında 
     "yourwebsite.com" kısmını kendi domain adınızla değiştirin
   - Sitemap.xml'deki tarihleri güncelleyin

3. 🔒 SSL SERTİFİKASI:
   - SSL sertifikası yükleyin
   - .htaccess dosyasındaki HTTPS yönlendirme satırlarının 
     yorumunu kaldırın (# işaretlerini silin)

4. 📧 İLETİŞİM FORMU:
   - script.js dosyasındaki form gönderme fonksiyonunu 
     gerçek bir API endpoint'i ile değiştirin
   - Alternatif: PHP mail() fonksiyonu veya üçüncü parti 
     servis (EmailJS, Formspree) entegrasyonu

5. 🔍 SEO YAPILANDIRMASI:
   - Google Search Console'a site ekleyin
   - Google Analytics kodu ekleyin (script.js'te belirtilen yere)
   - Sitemap'i Google'a gönderin
