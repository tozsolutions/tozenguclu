⚙️ BACKEND KURULUMU (Render):
1. GitHub repo oluştur
2. backend/ klasörünü push et
3. Render'da "New Web Service" oluştur
4. GitHub repo'yu bağla
5. Environment variables:
   - OPENAI_API_KEY: sk-your-api-key
6. Deploy et
7. URL'i chatbot.js'te güncelle

═══════════════════════════════════════════════════════════════════
                    BACKEND KURULUM DETAYI
═══════════════════════════════════════════════════════════════════

🔧 RENDER KURULUMU:

1. 📁 GITHUB REPO:
   - Yeni repo oluştur: "modernsite-ai-backend"
   - backend/ klasörünü yükle
   - requirements.txt, app.py, render.yaml dahil

2. 🌐 RENDER DASHBOARD:
   - render.com'a git
   - "New" > "Web Service"
   - GitHub'ı bağla
   - Repo seç: "modernsite-ai-backend"

3. ⚙️ AYARLAR:
   - Name: modernsite-ai-backend
   - Environment: Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: gunicorn app:app
   - Plan: Free ($0/month)

4. 🔐 ENVIRONMENT VARIABLES:
   - OPENAI_API_KEY: sk-your-openai-api-key
   - FLASK_ENV: production
   - FLASK_DEBUG: false

5. 🚀 DEPLOY:
   - "Create Web Service" tıkla
   - 5-10 dakika bekle
   - URL: https://modernsite-ai-backend.onrender.com

6. 🔗 FRONTEND BAĞLANTISI:
   - ai-version/chatbot.js dosyasını aç
   - Line 8: apiEndpoint'i Render URL ile değiştir
   - Örnek: 'https://modernsite-ai-backend.onrender.com/api/chat'

═══════════════════════════════════════════════════════════════════
                    OPENAI API KEY ALMA
═══════════════════════════════════════════════════════════════════

🔑 OPENAI SETUP:
1. platform.openai.com'a git
2. Hesap oluştur/giriş yap
3. "API Keys" sekmesine git
4. "Create new secret key" tıkla
5. Key'i kopyala (sk-... ile başlar)
6. Render'da Environment Variable olarak ekle

💰 MALİYET:
- GPT-3.5-turbo: $0.0015 / 1K tokens
- Aylık tahmini: $5-20 (kullanıma göre)
- İlk $5 ücretsiz krediniz var
