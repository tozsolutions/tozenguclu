🚀 NETLIFY'A YÜKLEME - 3 FARKLI YÖNTEMİ

═══════════════════════════════════════════════════════════════════
                    YÖNTEM 1: DRAG & DROP (EN KOLAY)
═══════════════════════════════════════════════════════════════════

1. 📁 DOSYALARI HAZIRLA:
   - Tüm dosyaları bir klasöre topla
   - Aşağıdaki dosyaların var olduğunu kontrol et:
   
   ✅ GEREKLI DOSYALAR:
   ├── index.html
   ├── styles.css  
   ├── script.js
   ├── netlify.toml
   ├── _redirects
   ├── thank-you.html
   ├── 404.html
   ├── favicon.svg
   ├── manifest.json
   ├── sitemap.xml
   ├── robots.txt
   └── sw.js

2. 🌐 NETLIFY'A GİT:
   - https://netlify.com adresine git
   - "Sign up" ile hesap oluştur veya "Log in" ile giriş yap

3. 🎯 DEPLOY ET:
   - Ana sayfada "Sites" sekmesine tıkla
   - "Add new site" > "Deploy manually" seç
   - Tüm dosyaları seçip sürükle-bırak yap
   - Otomatik deployment başlayacak (1-2 dakika)

4. ✅ TAMAMLANDI!
   - Site otomatik olarak yayına alınır
   - Size rastgele bir URL verilir (örn: https://amazing-site-123.netlify.app)

═══════════════════════════════════════════════════════════════════
                YÖNTEM 2: GIT REPOSITORY (ÖNERİLEN)
═══════════════════════════════════════════════════════════════════

1. 📁 GIT REPOSITORY OLUŞTUR:
   - GitHub, GitLab veya Bitbucket'ta yeni repo oluştur
   - Tüm dosyaları repo'ya push et

2. 🔗 NETLIFY'A BAĞLA:
   - Netlify'da "Add new site" > "Import an existing project" seç
   - Git provider'ını seç (GitHub/GitLab/Bitbucket)
   - Repository'yi seç
   - Build settings:
     * Build command: (boş bırak)
     * Publish directory: . (nokta)
   - "Deploy site" tıkla

3. 🔄 OTOMATİK DEPLOYMENT:
   - Her git push'ta otomatik deploy olur
   - Branch protection ile staging ortamı kurabilirsin

═══════════════════════════════════════════════════════════════════
                    YÖNTEM 3: NETLIFY CLI
═══════════════════════════════════════════════════════════════════

1. 💻 CLI KURULUMU:
   ```bash
   npm install -g netlify-cli
   netlify login
   ```

2. 🚀 DEPLOY:
   ```bash
   cd /path/to/your/website
   netlify deploy --prod --dir .
   ```

═══════════════════════════════════════════════════════════════════
                    DEPLOYMENT SONRASI AYARLAR
═══════════════════════════════════════════════════════════════════

🔧 SİTE AYARLARI:

1. 🏷️ SİTE ADI DEĞİŞTİRME:
   - Site dashboard > "Site settings"
   - "Change site name" ile özel ad ver
   - Örnek: yourcompany-website.netlify.app

2. 🌐 CUSTOM DOMAIN EKLEME:
   - "Domain settings" > "Add custom domain"
   - Kendi domain'inizi ekleyin
   - DNS ayarlarını yapın

3. 🔒 SSL SERTİFİKASI:
   - Otomatik olarak Let's Encrypt SSL verilir
   - "HTTPS" > "Force HTTPS" aktif edin

4. 📧 FORM NOTIFICATIONS:
   - "Forms" sekmesinde form gönderimlerini görürsünüz
   - Email notifications kurabilirsiniz
